Perguntas Frequentes

O que é HTML?

HTML é a linguagem de marcação usada para estruturar conteúdo na web.

O que faz a tag <details>?

A tag <details> permite esconder ou mostrar informações ao clicar no título, que é definido pela tag <summary>.
