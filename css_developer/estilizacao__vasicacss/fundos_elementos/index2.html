Exemplos de Background CSS

Background com Imagem

background-image: url('imagem.jpg');
background-size: cover;
background-position: center;

Gradiente Linear

background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

Imagem com Overlay

background: linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)),
url('imagem.jpg');

Gradiente Radial

background: radial-gradient(circle, #ff9a9e 0%, #fad0c4 100%);

Efeito Parallax

background-attachment: fixed;
background-position: center;
background-size: cover;

Padrão de Grid

background-image: linear-gradient(#e66465 1px, transparent 1px),
linear-gradient(to right, #e66465 1px, #f5f5f5 1px);
background-size: 20px 20px;

Múltiplas Imagens

background-image: url('img1.png'), url('img2.png'), url('img3.png');
background-position: left top, center center, right bottom;
background-size: 100px 100px;

Gradiente Animado

background: linear-gradient(-45deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
background-size: 400% 400%;
animation: gradient 15s ease infinite;

Textura Personalizada

background-image: radial-gradient(circle at 10% 20%,
rgba(255,255,255,0.2) 0%, transparent 20%),
radial-gradient(circle at 90% 80%,
rgba(255,255,255,0.2) 0%, transparent 20%);

Efeito Blur no Background

filter: blur(8px);
(aplicado a um elemento pseudo-background)
