Propriedades object-fit e object-position com Charmander

Demonstração usando a imagem de Charmander para mostrar como controlar o redimensionamento e posicionamento de elementos.

URL da imagem: https://wiki.otpokemon.com/index.php?title=Charmander

Propriedade object-fit

Como o Charmander se ajusta ao espaço disponível:

object-fit
fill
Padrão. O Charmander é esticado para preencher todo o espaço, podendo ficar distorcido. Não mantém proporções originais.
object-fit
contain
Mantém as proporções do Charmander. Aparece completo, mas pode deixar espaços vazios nas laterais ou topo/base.
object-fit
cover
Mantém proporções e preenche todo o espaço. Partes do Charmander podem ser cortadas para preencher o contêiner.
object-fit
none
Mostra o Charmander em seu tamanho original. Se for maior que o contêiner, será cortado; se for menor, mostrará com seu tamanho real.

Propriedade object-position

Controla qual parte do Charmander fica visível quando cortado:

object-position
center
Padrão. Centraliza o Charmander no contêiner. Mostra a parte central da imagem quando cortado.
object-position
left top
Alinha o Charmander no canto superior esquerdo. Mostra a cabeça e parte superior do corpo quando cortado.
object-position
right bottom
Alinha o Charmander no canto inferior direito. Mostra a cauda e parte inferior do corpo quando cortado.
object-position
80% 70%
Posiciona para destacar a chama da cauda do Charmander. Exemplo de posicionamento personalizado.

Diferenças Práticas

object-fit

  • Controla o redimensionamento da imagem
  • Define se mantém ou não as proporções
  • Decide se preenche ou corta a imagem
  • Útil para criar layouts consistentes

object-position

  • Controla o posicionamento dentro do espaço
  • Define qual parte da imagem fica visível
  • Trabalha em conjunto com object-fit
  • Essencial para enquadramento preciso

Exemplo Prático: Thumbnails de Pokémons

Uso combinado para criar uma grade de thumbnails uniformes:

.pokemon-thumb {
  width: 120px;
  height: 120px;
  object-fit: cover;
  object-position: top center;
  border: 2px solid #FFC107;
  border-radius: 8px;
}
