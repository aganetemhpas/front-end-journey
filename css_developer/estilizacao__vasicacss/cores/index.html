Cores no CSS

Existem várias formas de definirmos cores utilizando o CSS. Elas são:

  • Cores pré-definidas
  • Palavra-chave "currentcolor"
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

Dicas:

  • Passe o mouse sobre os itens para ver detalhes no código CSS
  • Experimente alterar os valores no código para ver diferentes resultados
  • Use ferramentas como o Color Picker do navegador para ajudar na seleção
